What is the eval() method?
The Eval function evaluates the string expression and returns its value. For example, Eval("1 + 1") returns 2.
What is the purpose of prev() and next() methods in jQuery?
The prev() method returns the previous sibling element of the selected element. next() method allows us to search through the immediately following sibling of these elements in the DOM tree and construct a new jQuery object from the matching elements.
How to delete all rows in a table except first row?
The jQuery function remove() removes all the rows except the first one.
