1.  What is container class in Bootstrap? When do we use container-fluid class?
Answer--Containers are the most basic layout element in Bootstrap and are required when using our default grid system.
	Use .container-fluid for a full width container, spanning the entire width of the viewport.

2.  What is CDN?  What are the advantages of CDN Urls?
CDN stands for Content Delivery Network. It is a geographically distributed network of proxy servers and their data centers.
Faster performance.
Reduce cost by caching.
Reliable as if any one server goes down, any other server from other location can send the data.
